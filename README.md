# Car Driving Simulation

This is a simple car driving simulation web app using HTML, CSS, and JavaScript.

## Features
- Real car image animation
- Road with continuous movement
- Keyboard and UI controls for acceleration, braking, and steering
- Speedometer and dashboard indicators

## How to Run Locally
1. Download and extract the files.
2. Replace `car.png` with your car image and `background.jpg` with a scenic background.
3. Open `index.html` in your browser.

## Host on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files.
3. Go to **Settings → Pages → Source → main branch**.
4. Your app will be live at `https://<username>.github.io/<repo-name>/`.
